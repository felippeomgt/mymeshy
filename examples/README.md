# Examples

Sample assets generated by MyMeshy, so you can see real output before installing
any models.

## `scimitar.glb`

A curved scimitar inspired by the dragon scimitar from Old School RuneScape,
generated locally with the **SDXL-Turbo → Hunyuan3D-2** text-to-3D pipeline on
a 12 GB RTX 3060.

- **Prompt:** *"A curved single-edged scimitar sword inspired by the dragon
  scimitar from Old School RuneScape: deep crimson-red and orange metallic
  blade with a sweeping recurved cutting edge, an ornate hooked crossguard, a
  short dark grip, and a rounded pommel. Polished fantasy game weapon prop."*
- **Output:** single-file GLB (~3 MB) with embedded PBR textures (albedo,
  normal, roughness, metallic, AO), ~30k triangles, normalized to a
  game-friendly scale.

Drop it into any glTF viewer ([gltf-viewer](https://gltf-viewer.donmccurdy.com/),
Blender, Godot, Unity, three.js) to inspect it.
